## QUICK START (5-MINUTE SETUP)

### Step 1: Choose Your Tool
- **Fastest:** Canva ($13/month)
- **Most Control:** Photoshop ($34/month)
- **Best for Scaling:** Figma (free/$12/month)

### Step 2: Define Brand Identity
1. Pick 2-3 primary colors (HEX codes)
2. Pick 2 fonts (one bold, one supporting)
3. Prepare logo file (PNG, transparent background)

### Step 3: Create 3 Templates
- Template A: Text-Heavy (40% of content)
- Template B: Visual-Heavy (40% of content)
- Template C: Balanced (20% of content)

### Step 4: Start Production
- 3-5 minutes per thumbnail
- 2-3 minutes if using Figma components
- Mobile test before uploading

## PRODUCTION WORKFLOW

### Weekly Production Schedule

**Monday: Planning (30 minutes)**
- [ ] Review CTR data from previous week
- [ ] Identify design patterns in top performers
- [ ] Plan 3-5 video thumbnails
- [ ] Assign template to each video (A/B/C)
- [ ] Gather images/video frames

**Tuesday-Thursday: Design (15-20 minutes total)**
- [ ] Batch design 3-5 thumbnails
- [ ] Use assigned templates
- [ ] Mobile preview each thumbnail
- [ ] Create A/B variations for high-potential videos
- [ ] Export at 1280x720 PNG

**Friday: Upload & Testing (10 minutes)**
- [ ] Upload all thumbnails to videos
- [ ] Set up A/B tests (2-3 variations per key video)
- [ ] Document in tracking spreadsheet

**Weekend: Review & Learning (20 minutes)**
- [ ] Review A/B test results
- [ ] Document learnings
- [ ] Update template variations
- [ ] Plan next week's iterations

**Total Weekly Effort:** ~75 minutes for 3-5 videos

### Per-Thumbnail Checklist

Before uploading ANY thumbnail:

**Technical Specs:**
- [ ] Dimensions: 1280x720 pixels
- [ ] Format: PNG
- [ ] File size: <2MB
- [ ] Aspect ratio: 16:9

**Design Principles:**
- [ ] 60-30-10 rule applied (60% hook, 30% brand, 10% text)
- [ ] Single focal point clearly visible
- [ ] Adequate white space
- [ ] No cluttered elements

**Color & Contrast:**
- [ ] Contrast score: 0.7+ (squint test passes)
- [ ] Maximum 3 saturated colors
- [ ] Color psychology aligned with content
- [ ] Mobile contrast verified

**Text Hierarchy:**
- [ ] 3-5 words maximum
- [ ] Font weight 700+ (Bold)
- [ ] Outline (3-5px) OR background box applied
- [ ] Readable at 160x90px (mobile)
- [ ] Brand font used

**Mobile Optimization:**
- [ ] Tested at 160x90 (mobile feed)
- [ ] Tested at 336x189 (tablet)
- [ ] Tested at 480x270 (desktop feed)
- [ ] Logo recognizable at all sizes

**Brand Consistency:**
- [ ] Logo in expected position
- [ ] Brand colors used
- [ ] Consistent with recent thumbnails
- [ ] Template version documented

**Competitive Check:**
- [ ] Distinct from similar video thumbnails
- [ ] Stands out in feed
- [ ] Hook element immediately clear

## QUICK REFERENCE GUIDE

### The 60-30-10 Rule
- **60%:** Visual Hook (main image, compelling visual)
- **30%:** Brand Elements (logo, color consistency)
- **10%:** Text (3-5 words maximum)

### The 50-Millisecond Rule
- Humans decide to click/skip in 50ms
- Design must grab attention immediately
- Simplicity > Complexity

### Text Hierarchy Rules
1. Font weight: 700+ (Bold, never regular)
2. Max words: 3-5 words total
3. Apply outline OR background box
4. Test at 160x90px (mobile display)

### Color Psychology Quick Reference
| Color | CTR Lift | Use Case |
|---|---|---|
| Red/Orange | +25-30% | Urgency, entertainment |
| Yellow | +20-28% | Surprise, education |
| Purple | +12-18% | Creativity, premium |
| Blue | +8-12% | Trust, professional |

### Mobile Sizes to Test
- 160x90px (mobile feed - PRIMARY)
- 336x189px (tablet)
- 480x270px (desktop feed)

### Contrast Score Quick Check
- Squint at thumbnail: can you still see main elements?
- If yes = sufficient contrast (0.7+)
- If no = increase contrast

## TROUBLESHOOTING

### Problem: CTR Below 3%

**Quick Fixes (In Order):**
1. Add white outline to text (5-minute fix)
2. Change text to warmest brand color available (5 minutes)
3. Increase primary image contrast (5 minutes)
4. Reduce text to 2 words maximum (5 minutes)
5. Test faceless visual hook (30 minutes)

### Problem: Inconsistent Thumbnails

**Solution:** Use templates religiously
- [ ] Canva: Create templates + use for every design
- [ ] Photoshop: Save layout template + lock brand layers
- [ ] Figma: Use components + document overrides

### Problem: Design Feels Overdone

**Quick Fixes:**
- Reduce colors from 4+ to 3 maximum
- Remove text outline if background is solid
- Use white space more effectively
- Simplify visual elements

### Problem: Text Unreadable on Mobile

**Quick Fixes:**
- Increase font size by 20-30%
- Add thicker outline (5-8px)
- Use solid background box instead of outline
- Reduce text to 2 words maximum
- Test at actual 160x90px size
